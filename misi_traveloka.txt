Requirement
1. Khusus pengguna ANDROID dan pengguna baru Traveloka
2. Download dan install hanya melalui link yang diberikan
3. Tidak berlaku bagi yang sudah pernah menginstal Traveloka
4. Berkesempatan mendapatkan evoucher Alfamart senilai Rp 10.000, jika kamu beruntung (akan dihubungi via WA).
Lakukan transaksi di Traveloka jika kamu ingin mendapatkan points.

Cara Pengerjaan
1. Download dan install aplikasi hanya melalui link yang ditentukan
2. Screenshot halaman profil Traveloka kamu
3. Dapatkan poin Traveloka jika kamu melakukan transaksi pembelanjaan

Rewards
Berkesempatan mendapatkan evoucher Alfamart senilai Rp 10.000, jika kamu beruntung (akan dihubungi via WA).
Lakukan transaksi di Traveloka jika kamu ingin mendapatkan points.
